# 🚀 Easy Code

> A modern and responsive web development portfolio built with **HTML5**, **CSS3**, and **JavaScript**.

Easy Code is a personal portfolio project created to showcase my learning journey in Web Development. The website features a modern cyberpunk-inspired design, Matrix-style animated background, multilingual support, responsive layout, and interactive user experience.
